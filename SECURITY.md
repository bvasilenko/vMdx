# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✓         |

## Reporting a vulnerability

Open a [GitHub Security Advisory](https://github.com/bvasilenko/vMdx/security/advisories/new) — do not file a public issue.

Include: affected version, reproduction steps, potential impact.

Response within 7 days. Patch + disclosure coordinated with reporter.
